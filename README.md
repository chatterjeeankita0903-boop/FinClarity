# FinClarity — Smart Finance Tracker

> AI-powered personal finance app for expense tracking, budgeting, and actionable insights.



**Live App:** https://fin-clarity-smart.lovable.app
**Built With:** Lovable · React · Supabase
**Supporting Material:** [Deck](https://1drv.ms/p/c/66052eb9d4e727fc/IQBdevdy_oTKRJW1UdV75Ad8AdZQ-_iQa7LJCukgMlqW9ik?e=7efetJ)

## Overview

FinClarity is an AI-powered personal finance app that brings expense tracking, budgeting, and financial insight generation together into a single, clean dashboard. Instead of juggling spreadsheets or multiple apps, users get one place to see where their money is going and what to do about it.

## Key Features

- Expense tracking across categories
- Budgeting tools to plan and monitor spending limits
- AI-generated, actionable financial insights surfaced directly in the dashboard
- Clean, single-screen view of overall financial health

## Context

Built as a rapid prototype to demonstrate how AI can turn raw transaction data into clear, personalized financial guidance — useful both as a standalone consumer concept and as a reference for AI-driven features in fintech products.

## Tech Stack

- **Lovable** — AI app builder used to scaffold and ship the working app
- **React** — front-end framework
- **Supabase** — backend/database for storing user and transaction data

## Running This Project

This is a hosted, live app — no installation needed to view it. If syncing the underlying code via Lovable's GitHub integration, clone the repo and run:

```bash
npm install
npm run dev
```

You'll also need a Supabase project connected (URL + anon key) for data persistence to work locally.
